# APEL-2025-2026-vente_jacinthes
Affiche pour la vente de jacinthes dans le cadre de l'APEL.

# Outils
Ordinateur sous Windows 11.

L'affiche a été créée avec Word de la suite office 365.

Le fond de l'affiche s'inspire du thème "Ready for Christmas XL" de slidesgo - voir la section credits. Le fond définitif a été créé par chatgpt en donnant un prompt demandant de créer une image aux dimensions d'une page A4 en gardant le style de l'image du thème.

Polices utilisées : "Assa fonts" et "KG Everywhere Everything" - voir la section credits

# Pour modifier l'affiche
Ouvrir le fichier source (APEL-2025-2026-vente_jacinthes.docx) avec Word. Les polices sont incluses dans le fichier source.

Pour une question pratique, je les ajoute également sur ce dépôt.

Respectez les crédits (slidesgo et les polices).

# Suppléments

Une version image PNG et une version PDF sont également fournies.
